# CTF Web Lab — Instruções dos Alunos

## Objetivo

Encontrar flags escondidas na API.

Formato das flags:

FLAG{exemplo}

## Regras

- Use apenas o ambiente fornecido.
- Não ataque sistemas externos.
- Não derrube propositalmente o lab.
- Documente cada descoberta.
- Explique o impacto da vulnerabilidade encontrada.

## Escopo

API local:

http://localhost:3000

## Entrega

Para cada flag encontrada, entregue:

1. Nome do desafio
2. Flag
3. Como encontrou
4. Impacto da falha
5. Como corrigiria
